# 白起 · 个人主页

> Vibe Coding 践行者 · GO / Java 全栈 · 偏后端 

交互式个人简历站，基于 React + Three.js 构建。通缉令造型的海报由 Verlet 积分粒子系统驱动，支持鼠标拖拽和周期性微风效果。

## 技术栈

- **React + Vite + TypeScript**
- **Three.js / React Three Fiber** — 3D 渲染
- **Verlet 积分** — 纸张物理模拟
- **Canvas 2D** — 动态海报纹理生成

## 本地运行

```bash
npm install
npm run dev
```
